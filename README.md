Supermarket Bill Generation System (Python)
Project Overview
This is a Python project that simulates a supermarket billing system.
You can choose items, enter the quantity you want to buy, and the program will generate a final bill. The bill includes the total amount, GST (5%), and the date and time.

Features
Choose items and enter the quantity
Automatic calculation of prices
GST calculation at 5%
Shows customer billing details
Displays date and time on the bill
Technologies Used

Python
Dictionaries and lists
Loops and conditional statements
Date and time module

How to Run
Copy or clone the repository using this link:
https://github.com/udaykirannaga/Supermarket-Bill-Generation-Python
Open the project folder on your computer.
Run the Python file supermarket_bill.py using your Python IDE or terminal.

Follow the instructions in the program:
Enter your name
Select the items you want to buy
Enter the quantity for each item
The program will generate your final bill with GST and total
